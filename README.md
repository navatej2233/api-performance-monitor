##  Description
The **API Performance Monitor** is a Python-based system designed to track and analyze API response times and status codes.  
It stores the metrics in an SQLite database and visualizes them in a live Flask web dashboard with Plotly charts.  
The dashboard automatically refreshes every 10 seconds to provide near real-time performance insights.
